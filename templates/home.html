About SAMS

SAMS is a sentiment analysis tool that can predict the sentiment of social media posts that uses a mix of Malay and English language in one sentence. This is achieved by training a machine learning model using various machine learning algorithms and a dataset of real social media posts, from which the best performing model was selected.

This simple web app was created to demonstrate the viability of the SAMS model on predicting real life social media posts. The app will scrape an Instagram page based on user input for the most recent comments, predict the sentiments of the comments and display a summary of the general sentiment on the page. This data will then be stored in a database to allow viewing at a later time.

Simply click here or the "Analyze Instagram Page" on the sidebar to start using the demo app. You can then enter the name of the Instagram page that you would like to see the sentiment of and the app will handle the rest for you.
